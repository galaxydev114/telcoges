# Telcoges
 Telcoges expenses & payments, invoices & estimats tracking system.
 
 ## Get Started
env setting

Yarn global install

Composer install
```sh
$ composer install
```

Laravel license
```sh
$ php artisan key:generate
```

Database migrate and seed
```sh
$ php artisan migrate --seed
```

Storage link
```sh
$ php artisan storage:link
```

Build Vue
```
yarn build
```

Run scropt
```
yarn dev
```
